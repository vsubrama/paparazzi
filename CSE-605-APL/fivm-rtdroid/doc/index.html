Quick Start Guide

To build Fiji VM, simply do: 
./configure
make
sudo make install
Now you should have the fivmc executable in /usr/local/bin, which should be in your path. To compile Java bytecode to native code, simply do: 
fivmc -o myProgram -m myMainClass <list of classes>
For example: 
fivmc -o myProgram -m myMainClass .
This will compile all of the classes found in the current directory, under the assumption that myMainClass is the name of the class with the main method. The result will be an executable program called "myProgram".
